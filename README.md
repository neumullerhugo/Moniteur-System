# Moniteur-System
Une appli pour remplacer la gestionnaire des tâches de windows.
Elle apparait sur votre burau pour vous eviter d'ouvrir le gestionnaire des tâches qui pour les moins avancer peut etre difficile à comprendre.

Comment s'en servir? : 
1. Une fois que vous l'avez télécharger et que vous l'ouvrez ,Windows va vous ouvrir une fenêtre :<img width="628" height="571" alt="Capture d&#39;écran 2026-07-03 004432" src="https://github.com/user-attachments/assets/59110faa-38c2-480c-8849-862a6de95fb5" />

2. Une fois que vous avez la pop-up cliquez "information complémentaire" et cliquez sur "éxecuter quand meme"
3. puis vous voula sur l'apppli Moniteur Systeme
